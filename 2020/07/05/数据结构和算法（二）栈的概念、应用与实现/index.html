栈 Stack

1、栈（Stack)的概念

  • 栈也是一种线性结构
  • 相比数组，栈对应的操作是数组的子集

注意： 栈只能从一端添加元素，也只能从一端取出元素。 用于存取的那一端称之为 栈顶。

2、入栈与出栈

​

入栈规则： 只能从栈顶将数据存入栈中。

出栈规则： 只能从栈顶将数据取出。

说明： 栈是一种后进先出的数据结构，英文称之为： Last In First Out(LIFO)

标记 : 在计算机的世界里，栈拥有着不可思议的作用

3、栈的应用

举例：

（1） 文本编辑器的撤销操作，其中使用的 Undo 操作（撤销） 其实现就是根据栈的原理达到的。

（2）以java代码为例，当多个method相互调用时，如A调用B，B调用C，此时从A方法出来去执行B的内容，会将Amethod中执行到的代码的行数记录到栈中，再将B method中执行到的行数记录到栈中，待执行完 method C后。再根据出入栈的规则获取到 method B中最近需要执行的那行代码的位置继续执行下去；然后再从栈中取 method A的执行记录。待method A中的代码执行完毕后，如果栈中没有数据，则说明该程序已执行完毕。

5.3 括号匹配 - 编译器 - 引入Leetcode问题

给定一个只包括 ‘(‘，’)’，’{‘，’}’，’[‘，’]’ 的字符串，判断字符串是否有效。

有效字符串需满足：

​ 左括号必须用相同类型的右括号闭合。

​ 左括号必须以正确的顺序闭合。

注意空字符串可被认为是有效字符串。

示例 1:
输入: “()”
输出: true
示例 2:
输入: “()[]{}”
输出: true
示例 3:
输入: “(]”
输出: false
示例 4:
输入: “([)]”
输出: false
示例 5:
输入: “{[]}”
输出: true

入栈规则：

一次将括号从右到左压入栈

出栈规则：

右侧的小括号与左侧的小括号匹配成功，则将左侧的小括号出栈；一次匹配中括号、大括号。

当栈中元素为空时，说明匹配成功了。也证明了该字符串是一个正确的有效括号的字符串。
