SELECT语句

SQL 语句构成

SQL语句是由简单的英语单词构成的。这些单词称为关键字，每个SQL语句都是由一个或多个关键字构成的。

一条查询语句必须有两点信息 —— 想选择什么，以及从什么地方选择。

检索不同的行

需求： 某些情况下，不希望将所有的结果都查询出来，只针对某个列结果不同的行进行返回。

解决方法： 使用 DISTINCT 关键字
