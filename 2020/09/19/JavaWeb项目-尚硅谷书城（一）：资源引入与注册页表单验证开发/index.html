JavaWeb项目-尚硅谷书城（一）：资源引入与注册页表单验证开发

发表于 2020-09-19 | 分类于 JavaWeb | | 阅读次数: | 阅读数

本节要点：

  • 创建静态项目工程，并引入相应的页面静态资源
  • 对注册页做表单验证的开发
    • 主要用到的知识点：jquery 语法 + 正则表达式
    • 验证逻辑：
      • 验证输入用户名、密码、手机号码、邮箱的格式是否符合规则，使用正则表达式验证。包括不能为空。
      • 密码与确认密码必须一致

1. 创建静态 Web 工程 —— book_city_project

2. 拷贝静态资源到static 目录下

static 目录下创建 script 目录，引入jquery-1.7.2.js

3. 在 pages/user/regist.html 注册页面进行注册逻辑代码的开发

具体根据以下的业务逻辑进行：

  • 验证用户名：必须由字母，数字下划线组成，并且长度为5到12位
  • 验证密码：必须由字母，数字下划线组成，并且长度为5到12位
  • 验证确认密码：和密码相同
  • 邮箱验证：xxxxx@xxx.com
  • 验证码：现在只需要验证用户已输入。因为还没讲到服务器。验证码生成。

注意： 要先引入 jquery 类库，不然会用 jquery 语法会报错。

4. 在 pages/user/login.html 注册页面进行注册逻辑代码的开发

具体根据以下的业务逻辑进行：

  • 验证用户名：必须由字母，数字下划线组成，并且长度为5到12位
  • 验证密码：必须由字母，数字下划线组成，并且长度为5到12位

注意： 要先引入 jquery 类库，不然会用 jquery 语法会报错。
