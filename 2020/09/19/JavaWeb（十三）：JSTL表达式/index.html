JSTL 标签库

1.什么是 JSTL 标签库

​ JSTL标签库 全称是指 JSP Standard Tag Library JSP标准标签库。是一个不断完善的开放源代码的JSP标签库。

​ EL表达式主要是为了替换jsp中的表达式脚本，而标签库则是为了替换代码脚本。这样使得整个jsp页面变得更佳简洁。

JSTL由五个不同功能的标签库组成。

功能范围 URI 前缀
核心**标签库**–重点 http://java.sun.com/jsp/jstl/core c
格式化 http://java.sun.com/jsp/jstl/fmt fmt
函数 http://java.sun.com/jsp/jstl/functions fn
数据库(不使用) http://java.sun.com/jsp/jstl/sql sql
XML(不使用) http://java.sun.com/jsp/jstl/xml x

在jsp标签库中使用taglib指令引入标签库

2. JSTL标签库的使用步骤

2.1 需要导入JSTL标签库的jar包

taglibs-standard-impl-1.2.1.jar

taglibs-standard-spec-1.2.1.jar

2.2 使用taglib指令引入需要的标签库

<%@ taglib prefix=*"c"* uri=*"http://java.sun.com/jsp/jstl/core"* %>

3. core核心库使用

i. <c:set />

1、可以往四个域中保存数据

2、还可以修改域中的bean对象的属性或是map集合中某个key的值

ii. <c:if />

​ if标签可以用来做if判断

iii. <c:choose> <c:when> <c:otherwise>标签

特别像switch case default}

iv. <c:url />

它可以根据给定的资源，工程路径，生成一个url地址输出

v.<c:forEach />

forEach标签就是做遍历操作。

  1. 遍历1到10，输出
  1. 遍历Object数组
  1. 遍历Map集合
  1. 遍历List集合—list中存放 Student类，有属性：编号，用户名，密码，年龄，电话信息
