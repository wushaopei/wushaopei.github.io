2、冒泡算法

​

平均时间复杂度：O(n2)

冒泡算法： 双重for循环，从小到大进行排序

  • 双重for循环意义：
  • 外层for循环： 代表循环的轮数，外轮每次遍历会从指针为0的地址开始if并交换，然后length-1，当进行i轮后，则数组的长度为length-i，数组中元素的索引为length-i-1
  • 内层for循环： 代表每一轮循环的次数，完整的数组中，arr[0]和arr[1]进行交换，然后arr[1]和arr[2]进行交换….，
  • 依次类推，这是每一轮遍历的次数，该次数受轮数变化后数组的length而影响，当经历i轮后，length随着减去i，这一部分是已经交换好的处在数组的末端，故j的指针不超过arr.length-i-1

4、选择排序

基本思想： 在长度为N的无序数组中，第一次遍历n-1个数，找到最小的数值与第一个元素交换；

第二次遍历n-2个数，找到最小的数值与第二个元素交换；

。。。

第n-1次遍历，找到最小的数值与第n-1个元素交换，排序完成。

​

平均时间复杂度：O(n2 )

解析：

​ 使用双层for循环实现选择排序

​ 1. 外层代表要循环的轮数，每一轮遍历一遍length-i长度的数组并获得其中的最小值

​ 2.定义最小值的默认指针

​ 3.内层for代表对当前这一轮的所有元素的遍历和比较，注意：j > i ,且满足指针 j 的值小于指针minIndex时才进行交换

​ 4.当 minIndex == i 时，说明指针 i 的值为最小值，不需要交换

5、插入排序

基本思想：
在要排序的一组数中，假定前n-1个数已经排好序，现在将第n个数插到前面的有序数列中，使得这n个数也是排好顺序的。如此反复循环，直到全部排好顺序。

过程：

​

平均时间复杂度：O(n2)

解析：

​ 插入排序在实现上，通常采用in-place排序（即只需用到O(1)的额外空间的排序），因而在从后向前扫描过程中，需要反复把已排序元素逐步向后挪位，为最新元素提供插入空间。

  1. 从第一个元素开始，该元素可以认为已经被排序； 故外层for循环的length-1
  2. 取出下一个元素，在已经排序的元素序列中从后向前扫描； 如果该元素（已排序）大于新元素，将该元素移到下一位置；
  3. 如果该元素（已排序）大于新元素，将该元素移到下一位置；
  4. 将新元素插入到该位置后；
  5. 重复步骤 2~5.
